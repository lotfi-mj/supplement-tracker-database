# Data Entry Guidelines

## Product Information

### Product IDs
- Use prefix based on category (VD for Vitamin D, MG for Magnesium, etc.)
- Follow with 3-digit number (001, 002, etc.)

### Units
- Use standard abbreviations:
  - IU (International Units)
  - mg (Milligrams)
  - mcg (Micrograms)
  - g (Grams)

### Forms
- Standard forms:
  - capsule
  - tablet
  - softgel
  - powder
  - liquid

### Certifications
- Use forward slash (/) to separate multiple certifications
- Common certifications:
  - USP
  - NSF
  - GMP
  - IFOS

## Categories

### Primary Categories
- vitamins
- minerals
- fish_oils
- herbs
- probiotics

## Quality Data

### COA Scores
- Use 0-100 scale
- Include verification date
- Document testing organization

### Overall Score Calculation
- 40% COA Score
- 30% Third Party Testing
- 30% Manufacturing Quality
